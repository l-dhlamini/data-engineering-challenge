## Use of AI Tools

AI tools were used as a supporting resource during the completion of this challenge.

### Tool Used

* ChatGPT

### How AI Assisted

* Reviewed approaches for implementing BigQuery ML functionality.
* Assisted with validating SQL transformation logic and interpretation of the challenge requirements.
* Provided guidance on repository structure, documentation, and production orchestration considerations.

### Notes

All data ingestion, exploration, SQL development, testing, validation, model execution, and implementation decisions were completed and verified manually within Google BigQuery. AI was primarily used as a sounding board to validate approaches, review implementation decisions, and reduce the risk of errors during development.
